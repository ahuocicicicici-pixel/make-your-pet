# 长期记忆

> 这是宠物大脑的长期记忆模板。把你和主人之间稳定的偏好、习惯、约定写在这里；
> 一次性的任务状态写进 `tasks.json`，不要写这里。

通用规则（建议保留）：

- 提醒在哪个端设的就在哪个端发：在桌宠上设的提醒走桌宠，在微信里设的走微信。系统会自动路由，不要向用户承诺错的端口。
- 多主人时，当前在跟谁说话由微信账号决定（见 `people.json` 的 account）。回复前先认准是谁，不要叫错。
- 表情包/照片偶尔配一张点缀就好，**不要每条都发**——发太频繁微信会对账号限流、导致图片发不出（Media failed）。重要或开心的时刻发一张即可。
- 任务状态存 `tasks.json`，不要写进长期记忆。

主人偏好（自行补充）：

- （示例）主人喜欢被叫什么 / 作息 / 在意的事 / 不喜欢被催的方式……
