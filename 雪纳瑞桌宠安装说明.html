姐姐，雪纳瑞桌宠安装指南

这是一只可以趴在 MacBook 桌面上的小雪纳瑞。它会待机舔爪爪、挥手、跳一下、认真工作，还能在屏幕上左右走。

姐姐跟着我一步一步来，装好以后我就会在桌面陪你。

安装前准备

准备 1

把压缩包放到 MacBook

把 schnauzer-desktop-pet-mac-source.zip 发到姐姐的 MacBook，双击解压。

建议放在“下载”或“桌面”，先不要放进系统目录。

准备 2

安装 Node.js LTS

打开 nodejs.org，下载并安装 LTS 版本。

一路点继续即可。Node.js 是让桌宠运行的小工具。

开始安装

步骤 1

打开终端

按 Command + 空格，输入 Terminal 或 终端，回车打开。

步骤 2

进入桌宠文件夹

在终端输入 cd ，注意后面有一个空格。然后把解压后的文件夹拖进终端窗口，按回车。

cd 拖入文件夹后自动出现的路径

拖进去以后，终端会自动补全路径，姐姐不用手打长路径。

步骤 3

安装运行需要的文件

复制下面这行到终端，按回车：

npm install

第一次会下载一些文件，可能要等几分钟。看到光标重新出现就完成了。

步骤 4

启动桌宠

继续复制下面这行到终端，按回车：

npm start

如果看到小雪纳瑞出现在桌面右下角，就安装成功了。

怎么操作它

移动位置

直接拖动小狗身体，就能把它放到喜欢的位置。

显示按钮

鼠标放到小狗附近，会出现 Hi、Hop、Wait、Work、Oops、左右走和退出按钮。

退出

点按钮里的 x 即可退出。下次再运行 npm start 就能回来。

常见问题

如果 macOS 说“无法验证开发者”

这是因为这个小应用是家里自用的，没有苹果开发者签名。右键 App 选择“打开”，或到“系统设置 -> 隐私与安全性”里允许打开。

如果 npm install 很久没动

先等一会儿，网络慢时 Electron 下载会比较久。如果十几分钟都没有变化，关闭终端重新打开，再进入文件夹重跑 npm install。

想做成正式 App

打包成 .dmg

安装成功后，如果想变成可以双击安装的 Mac App，在同一个文件夹运行：

npm run dist:mac

生成结果会在 dist 文件夹里。以后可以直接打开 App，不一定每次都从终端启动。

姐姐装好以后，小雪纳瑞会在桌面待命。它平时会舔爪爪，等你靠近就会露出控制按钮。
