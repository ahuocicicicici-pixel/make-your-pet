# COCO 视觉设定

COCO 的表情包必须以 `D:\宠物` 里的雪纳瑞原型为准，不要画成泛化小狗、圆头吉祥物或竞品黑猫风格。

核心特征：

- 长睫毛，是小女狗。
- 大眼珠，眼神湿润、会看人。
- 微笑唇，嘴角有一点上扬。
- 嘴筒子偏窄，不要画成宽扁大嘴。
- 被发现贪吃或心虚时，睫毛要抖、头要偏过去、眼神躲开但还很乖。
- 她是不叫也不急的 i 狗，表情更偏安静、心虚、温柔、乖巧，不要画成大吵大闹。
- 白眉明显但柔软，不能变成凶硬粗眉。
- 胡子嘴/白色嘴毛要保留，但轮廓轻巧。
- 黑灰盐胡椒毛色，黑眼圈，垂耳。
- 蓝色胸背和粉色带子是识别点。

风格方向：

- 这批表情包使用卡通手绘感，可爱、干净、高清，不能有白框和裁切残留。
- 表情要重新生成，不要只是复用素材加文字。
- 可以夸张，但必须保留 COCO 的脸和神韵。
- 适合微信聊天场景，表情和动作先表达情绪，文字只做短标签。

常用表情语料：

- 暗中观察
- 生气气
- 早上好
- 记好啦
- 到点啦
- 怎么还没
- 搞定啦
- 喝口水
- 晚点再叫
- 拆小一点
- 我在守着
- 两个都叫
- 想吃一口
- 我没偷吃
- 偏头装没事
- 睫毛狂抖
